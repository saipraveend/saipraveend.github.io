### Common Use Cases

```css
/* Primary action button */
background: var(--color-primary-500);     /* #0076A8 */
color: white;

/* Hover state */
background: var(--color-primary-600);     /* #006a99 */

/* Active/pressed state */
background: var(--color-primary-700);     /* #005980 */

/* Light background accent */
background: var(--color-primary-50);      /* #e6f4f9 */

/* Borders */
border-color: var(--color-primary-300);   /* #4db8dd */
```

## Accessibility - Contrast Ratios

All text colors meet WCAG AAA standards when used correctly.

### Text on White Background

| Color | Contrast | WCAG Level | Usage |
|-------|----------|------------|-------|
| `--color-neutral-900` (#1a202c) | **13.8:1** | AAA | Primary text ✅ |
| `--color-neutral-800` (#2f3842) | **11.7:1** | AAA | Headings ✅ |
| `--color-neutral-700` (#4a5361) | **8.9:1** | AAA | Body text ✅ |
| `--color-neutral-600` (#6b7784) | **5.8:1** | AA | Secondary text ✅ |
| `--color-primary-500` (#0076A8) | **4.7:1** | AA Large | Links, buttons ✅ |
| `--color-primary-700` (#005980) | **6.8:1** | AAA | Dark links ✅ |

### White Text on Colored Background

| Background | Contrast | WCAG Level | Usage |
|------------|----------|------------|-------|
| `--color-primary-500` (#0076A8) | **4.5:1** | AA | Buttons ✅ |
| `--color-primary-700` (#005980) | **6.9:1** | AAA | Dark buttons ✅ |
| `--color-secondary-500` (#ff9100) | **3.4:1** | - | Use dark text instead ⚠️ |
| `--color-accent-500` (#00c853) | **2.6:1** | - | Use dark text instead ⚠️ |

**Note:** Always use dark text on light/bright backgrounds (orange, green, yellow).

## Quick Reference

**Need a color? Use this flow:**

1. **Primary action?** → `--color-primary-500` (#0076A8)
2. **Text?** → `--color-text-primary` (#1a202c)
3. **Background?** → `--color-bg-primary` (#ffffff)
4. **Border?** → `--color-border-base` (#dce0e3)
5. **Success?** → `--color-success-base` (#28a745)
6. **Error?** → `--color-error-base` (#dc3545)
7. **Light accent?** → `--color-primary-50` (#e6f4f9)
